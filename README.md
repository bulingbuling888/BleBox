# 智慧数据分析平台 - 微信小程序

## 项目概述
本项目是一个基于微信小程序开发的智慧数据分析平台，主要用于通过蓝牙连接设备，实时接收、传输和分析各类传感器数据（如温度、湿度、ADC值、电压等）。平台提供了直观的数据可视化界面和数据管理功能，方便用户进行数据分析和查看。

## 功能特点
1. **蓝牙设备管理**：搜索、连接、断开蓝牙设备
2. **实时数据通信**：接收和发送蓝牙数据
3. **数据解析与处理**：解析蓝牙数据并进行过滤、统计分析
4. **数据可视化**：通过图表展示各类传感器数据的变化趋势
5. **数据管理**：支持数据导出、清空历史数据
6. **设备配置**：设置蓝牙UUID等参数
7. **自动发送**：支持设置数据自动发送周期

## 技术栈
- 前端框架：微信小程序原生框架
- 编程语言：JavaScript
- UI组件：微信小程序内置组件
- 图表库：微信小程序内置图表组件
- 数据存储：微信小程序本地存储

## 页面说明
1. **首页（index）**：负责蓝牙适配器的开启、关闭，设备搜索、连接和管理
2. **蓝牙通信页面（comm）**：负责与已连接设备的蓝牙数据通信，显示接收到的原始数据
3. **数据分析页面（data_analysis）**：展示解析后的各类传感器数据，提供图表可视化和数据统计功能
4. **设置UUID页面（setuuid）**：配置蓝牙服务UUID和特征值UUID
5. **关于页面（about）**：显示应用相关信息
6. **欢迎页面（hello）**：应用启动页面
7. **输出页面（out）**：数据输出页面

## 数据流程
1. 设备搜索与连接：通过首页搜索蓝牙设备并建立连接
2. 数据接收：蓝牙通信页面接收设备发送的数据
3. 数据解析：对接收的数据进行解析和过滤
4. 数据存储：将解析后的数据存储到全局数据中
5. 数据分析：数据分析页面从全局数据获取数据并进行统计分析
6. 数据可视化：通过图表展示数据变化趋势

## 使用说明
1. 打开微信小程序
2. 在首页开启蓝牙适配器
3. 点击"扫描设备"按钮搜索附近的蓝牙设备
4. 选择要连接的设备，点击连接
5. 连接成功后，点击"进入通信"按钮进入蓝牙通信页面
6. 在通信页面可以查看接收到的原始数据
7. 点击"数据分析"按钮进入数据分析页面查看图表和统计数据
8. 在数据分析页面可以导出数据、清空历史数据或切换图表类型

## 注意事项
1. 使用前请确保手机蓝牙已开启
2. 确保设备与手机距离在蓝牙通信范围内
3. 首次使用需要配置正确的蓝牙UUID
4. 数据导出功能将数据复制到剪贴板，请注意保存
5. 长时间不使用时，请断开蓝牙连接以节省电量
